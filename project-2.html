Project 02:
Unitu Now Live

Web Development and Design for Startup

Project Overview

A tech startup company based in UK, working with Universities all over England to build a stronger and well built student satisfication system to allow student feedback to be acted on much quicker, and in real time.

Project Goals

Design a clearer website, which explains to any new user what Unitu exactly is without having to find out from elsewhere. This being done by creating a clear landing page with infographics and well structred layout throughout.

Outcome

The outcome of the site resulted in an increase of new users by 5%, gaining heavy traction within the first few months.
